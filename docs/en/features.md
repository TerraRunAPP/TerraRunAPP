# Product features

English · [Русский](../features.md)

## Five pillars of TerraRun

### 1. Map and territory capture

- Closed GPS route → polygon on the map (outline follows the real track).
- Conflicts: overlap with another user’s zone is recalculated on the server — the overlapping area transfers to the new owner.
- Difficulty layers (beginner / advanced / expert).
- Zone customization: color and texture (Premium).
- Shared map for all users in a region.

### 2. Training and health

- Apple Fitness–style log: summary, plans, trends, exercises.
- **HealthKit** integration (running, walking, distance, heart rate).
- **8 running calculators** (pace, VDOT, heart rate, calories, etc.).
- Cloud session sync; trail flags — “run flags” on the map for other runners.

### 3. Social layer

- Activity feed, likes, comments.
- **Clubs** (including private — Premium): rankings, member mileage.
- **Events** and race catalog.
- Chats and push notifications.
- Public profiles, album, hero block on the map.

### 4. “Traveler” (World Journey)

- Meta-game: virtual route **Moscow → Vladivostok** based on accumulated distance.
- Friends on the map, club rankings, POI bonuses.
- Long-term engagement mechanics.

Learn more: [terrarun.ru/traveler.html](https://terrarun.ru/traveler.html)

### 5. AI coach “Mark Terry”

- Personal plan generation and completed workout analysis.
- Premium limits and quotas for Free users.
- Status: demo mode; full rollout — on the roadmap.
