## Collaboration formats

### Brands and sponsors

- **Sponsored POI** — point of interest on the “Traveler” map with a bonus for users.
- **Branded territory / challenge** — “Capture the zone around our store in a week.”
- **Co-branded Premium** — subscription with a partner promo code.
- **Media integration** — feed stories, hero block, push campaigns.

*What partners get:* reach among active runners, geotargeting, measurable KPIs (captures, distance, challenge participants).

### Running clubs and teams

- Official club page in the app.
- Club rankings and member mileage.
- Joint events and map-based group training.
- White-label onboarding for new members.

### Race organizers

- Event card with route, date, and registration.
- Trail flags and territories around start/finish.
- Push reminders for participants.

### Coaches and running schools

- Public coach profile, geotags for training spots.
- Schedule, service packages, client analytics.
- Visibility on the map in “Coaches” mode.

### Cities and corporations

- Corporate challenges.
- Employee / neighborhood activity map.
- Distance and engagement reports (aggregated, no personal data).

## Metrics for partners

| Metric | Description |
|--------|-------------|
| MAU / DAU | Active users |
| Captured territories | Count and area of polygons in a region |
| Distance | Total km for a club / challenge |
| Trail flags | Published run traces |
| Retention D7 / D30 | Retention after first capture |
| Premium conversion | Subscription conversion |
| Event funnel | Views → event registrations |

*Exact KPI set is agreed in a partnership agreement. Aggregated analytics — without personal data.*
