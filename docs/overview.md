# Обзор продукта

[English](en/overview.md) · Русский

**TerraRun** — мобильная экосистема для бегунов, где карта становится игровым полем: пользователи захватывают территории реальными маршрутами, оставляют следы пробежек, соревнуются в клубах и проходят мета-квест «Путешественник» по накопленной дистанции.

**Сайт:** [terrarun.ru](https://terrarun.ru)

---

## Проблема и возможность

| Боль рынка | Как TerraRun отвечает |
|------------|------------------------|
| Беговые приложения сильны в статистике, но слабы в эмоции и удержании | Игровая карта: территории, слои сложности, конфликты зон, достижения |
| Сообщества живут в Telegram и Instagram, а не в продукте | Клубы, лента, чаты, события и trail flags — внутри приложения |
| Брендам и организаторам сложно измерить офлайн-активность | Геопривязанные активности, события, POI-бонусы, аналитика клубов |
| Тренерам не хватает цифровой витрины рядом с картой | Каталог тренеров, геотеги, расписание, пакеты услуг |

**Целевой рынок:** Россия и СНГ — растущий сегмент любительского бега, parkrun-культура, клубные забеги, корпоративные и городские инициативы. TerraRun закрывает нишу **location-based fitness** с социальным слоем.

## Конкурентное позиционирование

| | Strava / Nike Run Club | TerraRun |
|---|------------------------|----------|
| Фокус | Статистика и соцсеть | **Карта + игра + сообщество** |
| Территории / AR-игра | Нет | **Да, с серверным конфликтом зон** |
| Клубы и события | Ограничено | **Встроено в продукт** |
| Мета-квест по дистанции | Нет | **World Journey** |
| ИИ-тренер | Ограничено / зарубеж | **Локальный стек** |
| Платформа | Закрытая экосистема | **API + партнёрские интеграции** |

TerraRun не заменяет Strava — он **добавляет игровой и локальный слой**, которого нет у лидеров категории.

## Почему сейчас

1. **Location-based fitness** — растущий тренд; бег — естественная активность для такой модели.
2. **Пустая ниша в РФ** — нет локального лидера с картой и территориями для бегунов.
3. **Готовый технический фундамент** — не концепт, а работающее приложение с облачной платформой.
4. **Гибкая партнёрская модель** — от одного POI до городского челленджа.
