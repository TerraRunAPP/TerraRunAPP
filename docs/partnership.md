## Форматы сотрудничества

### Бренды и спонсоры

- **Sponsored POI** — точка интереса на карте «Путешественника» с бонусом для пользователя.
- **Брендированная территория / челлендж** — «Захватите зону вокруг нашего магазина за неделю».
- **Co-branded Premium** — подписка с промокодом партнёра.
- **Медийная интеграция** — сторис в ленте, hero-блок, push-кампании.

*Что получает партнёр:* охват активной аудитории бегунов, геотаргетинг, измеримые KPI (захваты, дистанция, участники челленджа).

### Беговые клубы и команды

- Официальная страница клуба в приложении.
- Клубный рейтинг и километраж участников.
- Совместные события и тренировки на карте.
- White-label onboarding для новых членов.

### Организаторы забегов

- Карточка события с маршрутом, датой, регистрацией.
- Trail flags и территории вокруг старта/финиша.
- Push-напоминания участникам.

### Тренеры и школы бега

- Публичный профиль тренера, геотеги точек тренировок.
- Расписание, пакеты услуг, аналитика клиентов.
- Видимость на карте в режиме «Тренеры».

### Города и корпорации

- Корпоративные челленджи.
- Карта активности сотрудников / жителей района.
- Отчёты по дистанции и вовлечённости (агрегированные, без персональных данных).

## Метрики для партнёров

| Метрика | Описание |
|---------|----------|
| MAU / DAU | Активные пользователи |
| Захваченные территории | Количество и площадь полигонов в регионе |
| Дистанция | Суммарный км клуба / челленджа |
| Trail flags | Опубликованные следы пробежек |
| Retention D7 / D30 | Удержание после первого захвата |
| Conversion Premium | Конверсия в подписку |
| Event funnel | Просмотры → регистрации на событие |

*Точный набор KPI согласуется в партнёрском соглашении. Агрегированная аналитика — без персональных данных.*
