## Продукт в двух словах

TerraRun объединяет **карту, тренировки, социальную сеть и монетизацию** в одном iOS-приложении:

- **Захват территорий** — замкнутый GPS-маршрут превращается в полигон на карте; зоны могут пересекаться и «переходить» между бегунами.
- **Тренировочный дневник** — планы, тренды, HealthKit, беговые калькуляторы.
- **Сообщество** — лента, клубы, события, чаты, trail flags (следы пробежек на карте).
- **World Journey** — мета-квест «Москва → Владивосток» по накопленной дистанции.
- **ИИ-тренер «Марк Терри»** — персональные планы и разбор тренировок (в roadmap).

**Платформы:** iOS 17+ (SwiftUI) · облачный API · [terrarun.ru](https://terrarun.ru)
